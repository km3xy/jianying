2APK (lasuo.ai) 是一个在线工具，专门用于将 AI（如 ChatGPT、Claude、Gemini、DeepSeek 等）生成的代码或简单的 Web 项目快速转换并打包成 Android 应用 (APK 文件)。
​以下是该工具的核心功能和用法：
​1. 主要功能
​它的主要目的是降低开发门槛，让不懂 Android 开发的用户也能把 AI 生成的网页代码一键变成手机 App。
​AI 代码打包： 支持直接粘贴来自 AI 聊天界面的代码。
​支持多种格式： * 单文件 (Mode A)： 上传 .html、.js、.jsx、.ts 或 .tsx 文件。
​粘贴代码 (Mode B)： 直接把 AI 生成的代码粘贴进去。
​压缩包 (Mode C)： 支持上传来自 V0.dev、AI Studio 等平台的项目 ZIP 包（支持 React/Vite 项目）。
​自动转换： 它能自动识别 React 组件或标准的 HTML/JS 项目。
​2. 自定义选项
​在打包时，你可以进行简单的配置：
​App 名称和图标： 可以自定义生成的 Android 应用的名称和图标 (PNG/JPG)。
​权限设置： 默认开启网络访问（INTERNET），也可以手动添加摄像头、存储等权限。
​版本管理： 设置应用的版本号（默认 1.0.0）。
​3. 使用场景
​快速原型： 你用 AI 生成了一个好玩的游戏或工具网页，想在手机上像真正的 App 一样运行。
​Demo 展示： 开发者向客户演示 React 或 HTML 项目。
​轻量工具： 把常用的网页小工具本地化。
​4. 如何使用
​准备代码： 让 ChatGPT 或 Claude 为你写一个网页应用代码（比如一个计算器、待办事项列表或小游戏）。
​上传/粘贴： 打开 demo2apk.lasuo.ai，将代码粘贴进去或上传文件。
​配置： 设置你的 App 名字和图标。
​构建： 点击构建按钮，等待系统处理完成后，即可下载 APK 安装包并安装到 Android 手机上。
​注意： 生成的是 APK 文件，仅支持 Android 系统，不支持 iOS（iPhone）。
