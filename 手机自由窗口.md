✅ 一、开启全局侧边栏（最便捷）
 
1.打开「设置」→ 找到「更多设置」

2.进入「自由窗口」→ 开启「全局侧边栏」

3.勾选「全部场景中开启」，之后从屏幕左右边缘向内滑动即可呼出侧边栏
 
🪟 二、快速开启小窗的4种方式
 
1.最近任务开小窗
上滑并停顿进入多任务界面 → 长按应用卡片 → 点击「小窗」图标

2.应用内悬挂小窗
在应用内上滑并保持按住 → 当顶部出现「拖到此处悬挂」提示时，将应用拖入即可

3.侧边栏开小窗
滑出侧边栏 → 点击应用图标，直接以小窗形式打开

4.通知开小窗
长按通知卡片 → 下拉菜单 → 选择「小窗打开」对应应用
 
📏 三、小窗操作技巧
 
- 移动：按住小窗顶部的蓝条拖动位置

- 缩放：拖动小窗的边角调整大小

- 全屏：按住小窗底部的横线向下拉动

- 关闭：按住小窗底部的横线向上快速甩动

- 贴边迷你：将小窗拖到屏幕四角，它会自动缩成小图标，点击即可恢复
 
🖥️ 四、分屏模式（一屏两用）
 
1.进入多任务界面，长按应用卡片 → 拖到顶部「分屏区」

2.在下半屏选择第二个应用，即可完成分屏

3.也可以在侧边栏长按应用 → 选择「分屏」，快速配对应用
 
⚙️ 五、进阶设置
 
在「设置」→「更多设置」→「自由窗口」中，你还可以：
 
- 管理应用的小窗权限

- 调整小窗的透明度

- 自定义侧边栏显示的应用

1. 主要功能
它的主要目的是降低开发门槛，让不懂 Android 开发的用户也能把 AI 生成的网页代码一键变成手机 App。
AI 代码打包： 支持直接粘贴来自 AI 聊天界面的代码。
支持多种格式： * 单文件 (Mode A)： 上传 .html、.js、.jsx、.ts 或 .tsx 文件。
粘贴代码 (Mode B)： 直接把 AI 生成的代码粘贴进去。
压缩包 (Mode C)： 支持上传来自 V0.dev、AI Studio 等平台的项目 ZIP 包（支持 React/Vite 项目）。
自动转换： 它能自动识别 React 组件或标准的 HTML/JS 项目。
2. 自定义选项
在打包时，你可以进行简单的配置：
App 名称和图标： 可以自定义生成的 Android 应用的名称和图标 (PNG/JPG)。
权限设置： 默认开启网络访问（INTERNET），也可以手动添加摄像头、存储等权限。
版本管理： 设置应用的版本号（默认 1.0.0）。
3. 使用场景
快速原型： 你用 AI 生成了一个好玩的游戏或工具网页，想在手机上像真正的 App 一样运行。
Demo 展示： 开发者向客户演示 React 或 HTML 项目。
轻量工具： 把常用的网页小工具本地化。

配置： 设置你的 App 名字和图标。
